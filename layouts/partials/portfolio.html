Highlighted Research & Development

DNN model of T-Cell antigen encoding

A CNN model for capturing antigenicity observed by single cells, using high-throughput timeseries data of T-cell activation.

DeepLBC: DNN for ER+ breast cancer

Modified DNN based on ResNet50 and MaskRCNN for object detection on super-resolution microscopy data.

NeuroFX: Cloud neuro stimulation platform

A cloud-based platform for universal neural stimulation protocols using Pygame and Flask.

Vistraw: Optical Flow Neural Tracker

Python pipeline using OpenCV and RAFT model to track traveling waves in sensory visual pathways.

HiNTS: Early warning signal detection

Python toolbox for empirical dynamical reconstruction and anomaly detection in brain state activity.

GravPy: Gravitational waves extraction

Python toolbox for signal processing LIGO data using recursive Wiener filter techniques.
